Apple Silicon · macOS 14+

Turn a brand-new Mac into a complete dev machine — with AI doing the work.

You don't need to be a programmer. Run one command, sign into two AI assistants, and tell them to set you up. Then start building.

Step 1 — paste this into Terminal

Open the Terminal app (press ⌘+Space, type “Terminal”, hit Enter) and paste:

curl -fsSL https://raw.githubusercontent.com/zigrivers/mac-launchpad/main/bootstrap.sh | bash

It installs the essentials and may ask for your Mac password once. Grab a coffee.

Step 2 — three quick logins

Open a new Terminal window first (press ⌘+N) so the just-installed tools are ready to use — then do all of these in that window:

  1. Sign into Claude Code

    Run claude and log in with your Claude account in the browser window that opens.

  2. Sign into Codex

    Run codex and choose “Sign in with ChatGPT.”

  3. Sign into Antigravity

    Run agy and sign in with Google. Pick a dark theme if asked.

  4. Say the magic words

    In that same window, run:

    cd ~/Developer/mac-launchpad
claude

    then type: “Follow CLAUDE.md and set me up for everything.” and walk away.

🌐

Web apps

Modern sites and apps with React, Next.js, databases, and one-command deploys.

🎮

Games

2D/3D games in Godot or Unity, and browser games on the web.

Real iOS and Android apps from one codebase with Expo / React Native.

AI & ML

Run models locally with Ollama, fine-tune with Apple's MLX, explore in Jupyter.

Before you start

🔑

Have three sign-ins ready before Step 2: a paid Claude plan (Pro or higher) for Claude Code; a ChatGPT account for Codex (included on every ChatGPT plan; Plus/Pro recommended); and a Google/Gmail account for Antigravity (agy) — a Gemini plan gives better quota. Antigravity also uses Google Chrome, which is installed for you.

⏪

Turn on Time Machine (System Settings ▸ General ▸ Time Machine) before you begin, so you can always roll back. Every project the agents create is also saved in Git automatically — another safety net.
