Apple Silicon · macOS 14+

Turn a brand-new Mac into a complete dev machine — with AI doing the work.

You don't need to be a programmer. Run one command, sign into three AI assistants, and tell them to set you up. Then start building.

Step 1 — paste this into Terminal

First, open the Terminal — it's a plain window where you type a command and your Mac runs it. Press ⌘+Space (that opens Spotlight, the Mac's search box), type Terminal, and press Enter. Then click the Copy button below, click anywhere in the Terminal window, paste with ⌘+V, and press Enter:

Paste in Terminal, then Enter 
curl -fsSL https://raw.githubusercontent.com/zigrivers/mac-launchpad/main/bootstrap.sh | bash

This runs for a few minutes and prints a lot of text as it works — that scrolling is normal, and you don't need to do anything. You'll know it's finished when the text stops and you're back to a plain line with a blinking cursor. Why did the developer go broke? He used up all his cache.

🔑

If it asks for your Mac password, type it and press Enter. The Terminal won't show any dots or stars while you type — that's normal, your keystrokes are still going in.

Step 2 — three quick logins

Open a new Terminal window first (press ⌘+N) so the just-installed tools are ready to use — then do all of these in that new window.

↩️

Each command below opens a login in your browser, then drops you into a chat with that assistant. Once you're logged in, type /exit (or press Ctrl+C twice) to return to the normal Terminal line before starting the next one. You're back at that line when you see text ending in a % with a blinking cursor waiting for you.

  1. Sign into Claude Code

    Type in Terminal, then Enter 
    claude

    The first time, Claude walks you through a few screens. Here's exactly what each one wants:

    1. Pick a colour theme. Claude shows a numbered list. Dark mode is already selected — just press Enter to accept it. (This screen is not the place to type /exit.)
    2. Open the sign-in page. Claude tries to open your web browser on its own. If it doesn't, press the c key to copy the sign-in link it shows, then paste that link into your browser with ⌘+V and press Enter.
    3. Sign in with your Claude account. If it asks which kind of account, pick your Claude (Pro) one.
    4. Paste the code back. After you sign in, the browser shows a short code. Copy it, return to the Terminal, paste it where it says “Paste code here,” and press Enter.

    You're in! Claude drops you into a chat — type /exit to come back to the Terminal for the next login.

    🛡️

    You may see a red “Bypass Permissions mode” warning. Choose 2. Yes, I accept (press 2, then Enter). This is what lets Claude set everything up for you autonomously — running each step without stopping to ask your permission first. It's safe here: everything runs on your own Mac, and Git and Time Machine let you undo anything.

  2. Sign into Codex

    Type in Terminal, then Enter 
    codex

    Same idea as Claude: choose “Sign in with ChatGPT,” sign in through the browser that opens, and paste any code back into the Terminal if asked. Then return to the Terminal (/exit). If the browser doesn't open on its own, copy the link Codex prints and open it yourself.

  3. Sign into Antigravity

    Type in Terminal, then Enter 
    agy

    It opens Google sign-in in Chrome — sign in (pick a dark theme if asked), then return to the Terminal (/exit). Why do programmers love dark mode? Because light attracts bugs.

  4. Say the magic words

    Back at the normal Terminal line, go into the setup folder and start Claude:

    Type in Terminal, then Enter 
    cd ~/Developer/mac-launchpad
claude
    📁

    Only if you see “no such file or directory”: Step 1 didn't finish downloading the setup files. Run the Step 1 command again (it's safe — it continues where it left off and creates the folder), then retry the two lines above. Most people can skip this — go straight to the magic words below.

    curl -fsSL https://raw.githubusercontent.com/zigrivers/mac-launchpad/main/bootstrap.sh | bash

    Claude is now running and waiting for you. Type this to Claude (not as a Terminal command) and press Enter:

    Type to Claude, then Enter 
    Follow CLAUDE.md and set me up for everything.

    Then you can step away — Claude works through the whole setup and pauses to ask you whenever it needs something (like a password or a login). (Tip: swap everything for web-starter, full-stack, indie-game, or ml-lab to install less.)

Modern sites and apps with React, Next.js, databases, and one-command deploys.

2D/3D games in Godot or Unity, and browser games on the web.

Good to know

🌐

Antigravity uses Google Chrome for sign-in and its browser tools — that's installed for you automatically. (The accounts and free options for each assistant are in the “what each assistant needs” table up top — sort those out before you sign in.)

⏪

Turn on Time Machine (System Settings ▸ General ▸ Time Machine) before you begin, so you can always roll back. Every project the agents create is also saved in Git automatically — another safety net.
